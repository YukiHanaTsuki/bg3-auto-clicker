# 博德之门3 邪术师零元购自动交替连点器

一个轻量级鼠标交替点击工具，用于在《博德之门3》等游戏中两个固定位置之间自动交替点击。

## 功能

- 记录两个屏幕位置（F1 / F2）  
- 按 F3 在两个位置间交替点击，间隔可自定义  
- 启动后最小化到系统托盘，右键图标可打开设置  
- 支持修改热键和点击间隔

## 使用方法

1. 运行 exe（或安装 AutoHotkey v2 后运行 ahk 脚本）  
2. 将鼠标移动到第一个需要点击的位置，按 `F1` 记录位置  
3. 移动到第二个位置，按 `F2` 记录位置  
4. 按 `F3` 开始交替点击，再按一次停止
5. 按 `Esc` 退出程序  
6. 右键系统托盘图标 →「打开设置」可修改按键和间隔
7. 默认间隔400ms，个人觉得比较合适 

## 自定义说明

- 本工具使用 AutoHotkey v2 编写，源码见 `buy.ahk`  
- 配置自动保存在 `ClickerSettings.ini` 中
- 编译为 exe 需要安装 AutoHotkey v2 并使用 Ahk2Exe
